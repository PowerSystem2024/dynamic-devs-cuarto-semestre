# Game Project

Para correr el juego se deben seguir las instrucciones en termina:

```sh
cd game
python3 main.py
```

# App

Crear el entorno virtual y activarlo:

```sh
cd app
python3 -m venv venv
source venv/bin/activate
```

En el archivo requirements.txt se encuentran todas las dependencias del proyecto. Instalarlas con el siguiente comando:

```sh
pip3 install -r requirements.txt
```
